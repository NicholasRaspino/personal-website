Description

I created a static website using what I learned from the front-end portion of my web development course. The code is written in HTML and CSS and uses the Bootstrap front-end framework. I used Git for version control and hosted the website on GitHub pages using a custom url purchased from Google Domains.

The website serves as my portfolio for showcasing projects I have worked on while learning to become a developer. Additionally the website lists tools, links and learning resources by subject to give a clear picture of what I have studied so far. My goal was to organize everything I might need to set up a development environment on a new computer.
